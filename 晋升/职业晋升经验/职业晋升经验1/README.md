# 职业晋升经验1

- 工程师应日产出可量化成果
- 记录进步
- 积极沟通目标与需求
- 利用AI工具和经理支持提升解决问题能力。

工程师的工作习惯对于他们的工作效率和成就感有着重要的影响。其中，每天都要有一个可以量化的产出，例如提交一个Pull Request，这种习惯可以帮助工程师提高工作效率和成就感。分享工作上的经验和思考，也是帮助工程师成为Top1%的程序员的重要途径。

养成每天完成一些事情的习惯可以提升工作效率和创造力。建议记录每天的小进步和问题解决技能，这有助于提升内在动力和满足感。每天定一个小目标，充实自己，有明确的产出，这样的习惯可以帮助工程师提升工作效率和内在动力。同时，记录每天完成的微进步可以提升内在动力和创造力。主动了解客户需求和问题解决能力，可以提升职业发展和领导能力。

在职场中，工程师需要利用自己的技能解决问题。一款功能强大的多功能AI应用手机app——"hao AI ho"，可以帮助工程师更好地解决问题。在工作中，工程师需要与PM沟通取舍，解决问题的能力很重要。只看会不会某种语言的评价是片面的，需要看解决问题能力。让经理知道职业目标，利用资源，设定AGGRESSI目标，这也是工程师在职场中提升自己的重要方式。

如何与经理沟通，以达成职业目标，也是工程师需要注意的问题。主动与经理沟通，让他们了解你的目标，他们会给你更多的支持和反馈，帮助你实现目标。直接表达目标，得到直接的帮助，主动提出需求，让经理分派重要任务和反馈，利用经理帮助你实现职业目标，主动沟通很重要。
